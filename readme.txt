== Description ==

Allows you to add a widget to your site that contains information about a Glitch.com profile.
It is very basic at the moment, but you can edit the plugin yourself, if you want to.

== Installation ==

This section describes how to install the plugin and get it working.


1. Upload all the files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place the widget in your theme and adjust the Glitch ID to match yours
4. You might need to CHMOD the /cache/ folder to make it writable
